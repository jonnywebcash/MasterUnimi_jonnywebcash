La seguente cartella contiene lo svolgimento del progetto per il corso di Python del master in Data Science presso Unimi

Il progetto affrontato è "P6: I mammiferi depongono le uova?", esso è un problema di clustering su un dataset basato su animali.

La cartella contiene:
- main.ipynb: il file jupiter contenente il codice e la descrizione passo passo di come si è affrontato il problema
- main.py: lo script python usato per lo sviluppo delle funzioni e delle elaborazioni anche quando non si era connessi alla rete (contiene la stessa struttura codice del file main.ipynb ma senza dilungarsi in siegazioni testuali)
- support_function.py: lo script python con tutte le funzioni di supporto, ha permesso di rendere i file "main" più leggibili e scorrevoli
- la folder Dataset: al cui interno ci sono i file "zoo_data.csv" contenente il dataset da clusterizzare (faremo uso solo di questo file) e il file "class.csv" contenente l'associazione di ogni animale ad una determinata specie
- progetti.pdf: pdf con tutti i progetti tra cui poter scegliere
- README.txt: file .txt con la spiegazione del repository
- project_P6_FCP: la versione pdf di main.ipynb, contiene lo stesso testo con una selezione di immagini rispetto al file jupiter

Approccio al repository
Dentro la cartella dataset ci sono i dati utilizzati per il progetto, se una persona desiderasse partire per un suo lavoro personale (unito a progetti.pdf) li ha tutte le informazioni che gli servono per svolgere l'elaborato in autonomia

Si consiglia di aprire in parallelo la presentazione "project_P6_FCP.pdf" e il file jupiter "main.ipynb", se non si è connessi ad internet anche main.py svolge lo stesso scopo, così da poter leggere come è stato affrontato il problema di clustering e poter eseguire il codice direttamente blocco per blocco con gli eventuali grafici aggiuntivi
nel file support_function.py le funzioni hanno nomi "parlanti", come le variabili utilizzate per rendere più facile la compressione della loro funzione.
